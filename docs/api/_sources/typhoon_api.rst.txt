Welcome to Typhoon APIs' documentation
======================================

:Author: Typhoon HIL
:Release date: |today|

Typhoon APIs are a collection of Python packages which let you programmatically 
interact with your HIL device and Typhoon HIL software tools during runtime.

Documentation on each API package and possible function calls can be found in the tree below. 
Additionally, there is a separate documentation for TyphoonTest API available locally or 
`online <https://www.typhoon-hil.com/documentation/typhoon-hil-typhoontest-library/>`_
