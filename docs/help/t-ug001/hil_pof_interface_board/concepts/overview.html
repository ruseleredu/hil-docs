Overview

Overview of the purpose and connectivity of the HIL POF Interface Board

The POF Interface Board (Plastic Optical Fiber Interface Board) is a signal conditioning board used to convert voltage digital signals (inputs and outputs to Optical signals). The purpose of the POF interface is to provide a connection between Typhoon HIL simulator devices and user equipment using plastic optical fibers. In this way isolation issues, electrical noise, parasitic inductance, and the capacitance of the copper cable can be neglected.

The main field of application is in power electronics converters where gate driver firing pulses are usually connected through POF cables. The board contains 32 Optical inputs and 32 Optical outputs. It is based on Avago optical receivers and transmitters with 50 Mbaud signal rates. All signals are also available on output connectors as CMOS level digital signals. Signal Logic is OR, which means that if optical input is used, electrical input shall leave floating.

Important: HIL Analog channels and user power supplies are Pass-Through and not used on the board.

The board features the following key sections, as shown in Figure 1.

  1. HIL Analog connectors
  2. HIL Digital connectors
  3. Digital to optical conditioners
  4. Optical IO connectors
  5. Digital IO passthrough connectors
  6. Analog IO passthrough connectors
  7. External DC power supply (9 - 24 V)
