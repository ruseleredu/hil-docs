Firmware update

Firmware update guide for the HIL Remote IO Interface

To update the HIL Remote IO Interface, run the update script located in the THCC installation directory: <installation path>/bin/remote_io.cmd
Note: This script can be executed only on Windows.

Ensure that the Remote I/O device is connected to the same network as the PC running the script.

Each Remote I/O device uses a static IP address in the format 192.168.20.X, where X = (Device ID + 1).

Example: If the Device ID (back of the device) is 2, the IP address will be: 192.168.20.3.
