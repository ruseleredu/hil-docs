Supported cards

A list of supported TI Launchpad cards

Note: Due to a possible collision with the BOOT MODE pin, some TI Launchpad cards can be used as they are, while some must be modified.

Cards supported without modification

The HIL TI Launchpad Interface is designed to support the following TI Launchpad cards, without any modification to the Interface or Launchpad:
  • LAUNCHXL-F28027F
  • LAUNCHXL-F28069M
  • LAUNCHXL-F28377S
  • LAUNCHXL-F28379D

Cards supported with modification

Due to a collision of the HIL Digital Output stage with the BOOT MODE pin, one digital buffer IC on the HIL Launchpad Interface needs to be electrically disabled, by modifying the ROE1 and/or ROE2 solder jumper position, from right (default) to left.
