TyphoonSim license

This document describes the TyphoonSim license type.

TyphoonSim license type

The TyphoonSim license is a user-based license. This means that the license is tied and granted to a specific user account on the Typhoon HIL platform. The same account is used also to access the ticketing system, forum, subscription, etc. Hence, a single account is used for all Typhoon HIL-related resources.

Since the license is user-based, you have to log in to the Typhoon HIL software environment to verify your identity and license. The TyphoonSim license is downloaded automatically once you login to the software. Logging in to TyphoonSim Software contains a guide on how to login. As the license for TyphoonSim is user-based, you can use the same license on more than one PC.

If you use TyphoonSim within Typhoon HIL Control Center, the license that you get for TyphoonSim will not override or affect in any way your existing license for HIL devices in Typhoon HIL Control Center.

How to get a TyphoonSim license

If you want to get a licence for TyphoonSim, please check Getting your License, or visit the Typhoon HIL website where you can apply for a license. If you are an Academic customer, a license will be automatically granted after applying. If you are Commercial customer, you can get a free trial license before purchasing a TyphoonSim license.

TyphoonSim license pricing

The TyphoonSim license for academic use is free of charge.

For the price of the TyphoonSim license for commercial use, please contact Typhoon HIL.

Free and trial license limitations

TyphoonSim license for academic customers is free, but it includes complete industry grade features. It has no limit on the feature set, nor the model size. The same is valid for the free trial license for commercial customers.
