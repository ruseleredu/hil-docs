HIL506 Device Configuration Table

Description of the HIL506 Device Configuration Table, which explains the HIL device resources available for different models and different configurations on the HIL506 device.

Real-time only: This document is valid only for real-time/VHIL simulation.

The Device Configuration Table provides information about the FPGA solver features available in different firmware configurations. It is currently available through the Device Manager tool via the Device table button. This button is also accessible in Schematic Editor under Model->Model Settings. You can set your Device and Configuration ID in Schematic Editor in the Model->Model Settings dialog.

The HIL506 Device Configuration Table is described in Table 1. For more information about what each line represents, please refer to the Device Configuration Table documentation.

Note: Some device configurations may be limited depending on the conditions of your software license. For more information about the configurations available with your license, please refer to the Subscription portal or contact your Sales representative.
Note: Custom configurations for HIL devices are also possible via the Custom Firmware Configurator tool, available in the Subscription portal. You can find more information as well as a tutorial video on this page.
