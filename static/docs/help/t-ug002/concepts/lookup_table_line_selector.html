Lookup table - Line index selector

Overview of the Lookup table line index selector and how you can interact with it.

The Lookup table line index selector is a panel for selecting the current active line, for creating new lines, and for removing unnecessary ones.

Changing the selected line will trigger an update in other panels:

  • Lookup table - Preview - Table will be repopulated with data from the newly selected line
  • Lookup table - Chart area - All line-specific actions will now affect the newly selected line

Both creating and removing a line are reversible actions stored on the undo stack.
