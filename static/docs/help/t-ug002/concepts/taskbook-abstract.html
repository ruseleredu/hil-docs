Abstract

Typhoon HIL Software Manual provides a detailed description of all features and functionalities available in various software tools which can be launched from Typhoon HIL Control Center: Typhoon Schematic Editor (including the Schematic Editor Component Libraries), HIL SCADA, Waveform Generator, TI Control Panel, HIL Firmware Manager, and Test and Calibration Tool. The Typhoon HIL Software Manual also provides information on the minimum system requirements needed to run Typhoon HIL software and step-by-step instructions on how to install the HIL USB and Ethernet Drivers.
