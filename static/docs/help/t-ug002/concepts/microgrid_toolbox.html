Microgrid Toolbox

Schematic Editor functionalities available through the Microgrid Toolbox

Real-time only: This document is valid only for real-time/VHIL simulation.

The Microgrid Toolbox is an additional toolbox containing support for real-time modeling of a growing collection of ubiquitous microgrid device models. Its main purpose is to enable you to accelerate the process of building microgrid models for simulation and analysis, allowing you to fully concentrate on the system and/or device under test, be it a microgrid controller, a DER controller, a protection relay, or another device. The Microgrid Toolbox enables early tests of microgrid controllers, communication protocols, device-to-device interactions, and system design with the use of complete models of DERs, loads, protection devices, gensets, and many others. These models can be executed in a great range of tests that cover different operation scenarios of a Microgrid.

Table 1 lists all Schematic Editor components available for real-time modeling through unlocking the Microgrid Toolbox. You can find these components in the Schematic Editor Microgrid Library.

For information about other toolboxes, please refer to the Schematic Editor Toolboxes documentation.
