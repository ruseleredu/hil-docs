Detailed description

Detailed description of the functionality of the HIL Connect.

The standard HIL Connect consists of:
  • Chassis with 8 slots
  • Up to 8 cards
The system is designed to be user-upgradeable, with each card being removable by undoing two flat-head screws and using top and bottom latches for easy extraction.

Each HIL Connect is uniquely defined by its Configuration Code, which encodes all relevant information, such as:

  • backplane revision
  • type, position, revision and hardware settings for each card slot
