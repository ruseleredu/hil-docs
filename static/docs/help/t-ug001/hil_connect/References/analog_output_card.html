Analog Output card

Description of the Analog Output card for HIL Connect hardware

Note: For more information on how to configure your card in Typhoon HIL Control Center, please refer to the Analog Output Card parametrization documentation.

The Analog Output card offers 32 analog outputs linked to 32 HIL AO channels. Each output can be configured for voltage or current mode (supported only in revisions 2.4 and later).

Analog Output card revisions

Table 1. Card revisions
Code Revision Part number
NI 2.5 26301
LU 2.4 24569
JT 2.3 23268

Analog Output card hardware specification

Table 2. Hardware Specification
Parameter Value
Type Analog output
Number of channels 32
Input signal range ±10 V
Output signal range ±10 V or ±20 mA
Load range >500 Ω (voltage mode) or <500 Ω (current mode)
Bandwidth (-3dB) DC to 100 kHz
Accuracy 0.1% + 500 uV (voltage mode) or 0.1% + 10 uA (current mode)
Isolation No

Analog Output card hardware settings

Table 3. Hardware Options
Option Method Possible options
Voltage or current mode selection Via internal DIP switches Each channel can be set to voltage or current mode
