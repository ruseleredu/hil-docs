Fiber Optic Input card

Description of the Fiber Optic Input card for HIL Connect hardware

Note: For more information on how to configure your card in Typhoon HIL Control Center, please refer to the Fiber Optic Input Card parametrization documentation.

The Fiber Optic Input Card offers 16 fiber optic input channels linked to 16 HIL DI channels, designed to support both high and low bandwidth applications. The Fiber Optic Input card accepts industry-standard 1 mm POF, with lengths of up to 50 m.
