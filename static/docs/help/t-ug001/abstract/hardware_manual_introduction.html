Hardware Manual Introduction

Typhoon HIL produces Typhoon HIL simulator devices, HIL Interfaces, and HIL Accessories designed to support real-time testing with your control equipment. The following sections feature a collection of some of the guides on the deployment, use, and management of Typhoon HIL hardware devices.

Note: Currently, not all hardware documentation is available for web access. If a guide you need is missing here, please contact us.

HIL Interfaces Online Documentation

Table 2. Summary of HIL Interface documentation and guides available online
Category Description of Content

HIL dS Interface Type-B Information about the HIL dS Interface Type-B, an interface board which allows for connecting dSPACE hardware to Typhoon HIL simulators.
HIL dS Interface Type-C Information about the HIL dS Interface Type-C, an interface board which allows for connecting dSPACE hardware to Typhoon HIL simulators.

HIL Remote IO Interface Information about the HIL Remote IO Interface, a device for IO expansion in HIL test setups.
HIL Power Link Interface Information about the HIL Power Link Interface designed for converting electrical signals to optical signals, and vice versa.
HIL DSP 180 Interface Information about the HIL DSP 180 Interface, an interface board which allows for connecting Texas Instruments Control Card boards to Typhoon HIL simulators.
HIL TI Launchpad Interface Information about the HIL TI Launchpad Interface, an interface board which allows for connecting TI's LaunchPad™ boards to Typhoon HIL simulators.
HIL TI uGrid Launchpad Interface Information about the HIL TI uGrid Launchpad Interface, an interface board which allows for connecting up to three TI LaunchPad™ boards to Typhoon HIL simulators.
HIL Infineon TriBoard Interface Information about the HIL Infineon TriBoard Interface, an interface board which allows for connecting Infineon TriBoard evaluation boards to Typhoon HIL simulators.
HIL Nucleo-64 Interface Information about the HIL Nucleo-64 Interface, an interface board which allows for seamlessly connecting STM Nucleo-64™ development kits to Typhoon HIL simulators.
HIL POF Interface Board Information about the HIL POF Interface Board, an interface board which allows for converting HIL voltage digital signals to Optical signals, and vice versa.
HIL CCS Interface Information about the HIL CCS Interface, a dedicated module for testing smart charge communication of electric vehicles and electric vehicle supply equipment.
HIL Electric Vehicle Charging Interface Information about the HIL Electric Vehicle Charging Interface solution, a dedicated module for testing and validating Electric Vehicle Supply Equipment (EVSE). This interface requires a connected power amplifier to function.
HIL Regatron Interface Device Information about the HIL Regatron Interface Device, an interface which allows for seamless communication between Regatron Power Source devices and Typhoon HIL simulators.
