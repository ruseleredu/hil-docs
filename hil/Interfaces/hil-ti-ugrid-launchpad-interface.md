## [HIL TI μGrid LaunchPad Interface](https://www.typhoon-hil.com/products/hil-interfaces/hil-ti-interface-cards/hil-ti-%ce%bcgrid-launchpad-interface/) {/* #hil-ti-μgrid-launchpad-interface */}

The HIL TI Launchpad Interface provides a pin-to-pin compatible interface between Typhoon HIL emulators and up to three C2000 Launchpad boards. In less than 5 minutes you can plug a TI Launchpad into a Typhoon HIL real-time emulator, map the signals, and run models with closed loop control.
