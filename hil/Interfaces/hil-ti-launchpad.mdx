## HIL TI LaunchPad Interface {/* #hil-ti-launchpad-interface */}

Typhoon’s HIL TI Launchpad Interface is a specifically designed interface board, to enable seamless Plug and Play integration between TI’s LaunchPad™ development kits for C2000™ Delfino MCU’s and any of the HIL4 and HIL6 series real-time simulation devices.


The Typhoon’s HIL TI Launchpad interface board has been optimized for the following TI C2000 LaunchPad cards:

- [LaunchXL-F28027F](https://www.ti.com/tool/LAUNCHXL-F28027F)
- [LaunchXL-F28069M](https://www.ti.com/tool/LAUNCHXL-F28069M)
- [LaunchXL-F28377S](https://www.ti.com/tool/LAUNCHXL-F28377S)
- [LaunchXL-F28379D](https://www.ti.com/tool/LAUNCHXL-F28379D)
- LaunchXL-F280025C
- LaunchXL-F280039C
- LaunchXL-F280049C
- LaunchXL-F2800137
- LaunchXL-F2800157
- LaunchXL-F28P55X
- LaunchXL-F28P65X
